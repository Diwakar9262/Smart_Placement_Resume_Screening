# Smart Placement & Resume Screening Engine

## Overview

This project is an AI-powered Resume Screening System developed using Python. It simulates a simplified Applicant Tracking System (ATS) that stores candidate information, analyzes resumes, predicts resume scores using Machine Learning, ranks candidates, and provides hiring insights.

## Features

* Add Candidate
* Update Candidate
* Delete Candidate
* Search Candidate
* Export to CSV
* JSON Data Storage
* Candidate Ranking
* AI Resume Score Prediction
* Skill Match Percentage
* Missing Skills Detection
* Learning Recommendations
* Dashboard Summary
* Candidate Filters
* Search by Skill
* Search by Multiple Skills
* Top N Candidates
* ATS Project Report
* Input Validation
* Exception Handling

## Technologies Used

* Python
* Object-Oriented Programming
* JSON
* CSV
* Pandas
* NumPy
* Scikit-learn
* Linear Regression
* Git & GitHub

## Machine Learning

The project uses Linear Regression to predict Resume Scores based on candidate Skills and Age.
